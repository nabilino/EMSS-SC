Utility Log Selection Criteria
  • Show Me:
    • Selecting All User ID Maintenance, All System Maintenance, or All ASP Tools will produce a report consisting of the subset of functions listed below each option.
    • Selecting a specific function will produce a report consisting only of activity for that function.
      • Example: Select EDI Maintenance to produce a report of all EDI Maintenance activity. This will include stopping and starting the EDI CommServer and the EDI E-Mail Notifier along with changes to the E-Mail notification list.

  • From Date:
    • Date must be in mm/dd/yyyy format
    • Select the to display a calendar you can use to select the date.

  • To Date:
    • Date must be in mm/dd/yyyy format
    • The To Date field must be greater than or equal to the From Date field.
    • Select the to display a calendar you can use to select the date.

  • Performed By:
    • Enter a User ID to produce a report of the selected function performed by that user.

  • Message Text Search:
    • Message Text Search can be used to find an EXACT match (case insensitive) of a string in the message text.
      • Example: Enter a user id to produce a report of any activity done to that particular user id.
      • Example: Enter "web server" when selecting "Add User" to produce a report of only Web Server Add User activity
