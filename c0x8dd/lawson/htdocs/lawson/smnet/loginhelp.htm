Denver Health Lawson Portal Login Help

If you are using Self Service for the first time, your user ID is your initials in lowercase letters followed by your employee number. Your password is your initials in lowercase and the last six digits of your social security number.

For example, if John Doe's employee number is 12345 and his social security number is 987-65-4321, his user ID is jd12345 and his password is jd654321.

If you do not know your user ID or password, please contact the Customer Support Center using the phone number below :

1-855-555-5555 (1-855-555-5555)

When entered incorrectly, the ID and password text boxes turn yellow. After five attempts, your user ID will be locked (you will see a blue screen) and you will need to contact the Customer Support Center.

The following requirements exist to access the Lawson Portal and Employee Self Service :

Operating System requirements :

  • Windows XP Pro
  • Windows Vista Business Edition
Web Browser requirements :
  • Internet Explorer 6.0 or above
  • MSXML 4.0 SP2 (4.20.9828.0 or higher)
The following operating systems and web browsers are not certified or supported with Portal :
  • Macintosh OS
  • Netscape
  • Firefox

NOTE: The system is closed during payroll processing. The payroll process begins on Tuesday evenings and is generally complete by Wednesday afternoon the week of your pay date. You cannot access the system during this time.

The Employee Self Service application uses pop-up windows. If you have a pop-up blocker, you may need to disable it.
