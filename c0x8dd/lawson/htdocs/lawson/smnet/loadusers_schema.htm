Mass Add

The CSV file must contain headers. The header line must contain the required RM attributes.
Each employee record must have a value for required RM attributes.
Required attributes in red

  • userid
  • osid n/a for Self Service
  • ospasswd
  • fname
  • lname
  • checkls Values YES or NO
  • role Separate multiple roles with commas, for example, Role = "Role1,Role2".
  • email
  • access
  • group Separate multiple groups with commas, for example, Role = "Group1,Group2".
  • portaladmin Y or N
  • addins Y or N
  • wfuser 0 or 1
  • prodline
  • portalrole
  • essservice If creating Employee Self Service IDs this attribute is required
  • company
  • employee
  • rssservice If creating RQC Self Service IDs this attribute is required
  • requester
  • vssservice
  • vendor
  • vendorgroup
  • cssservice
  • customer
  • customergroup
  • fssservice
  • activitylist
  • acgrpcollect
